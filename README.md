A modern and user-friendly **BMI (Body Mass Index) Calculator** built using **Python and Tkinter**.
This application allows users to calculate their BMI and instantly know their health category with a soft pastel UI.

##  Features
* Simple and clean graphical interface
* Takes height and weight as input
* Calculates accurate BMI score
* Displays health category (Underweight, Normal, Overweight, Obese)
* Shows helpful health tips
* Reset and exit options

##  Technologies Used
* Python
* Tkinter
* Datetime (for logical operations)

##  Installation
1. Clone the repository:
git clone https://github.com/Satvika-Moota/oibsip_python_task2

2. Go to the project folder:
cd BMI-Calculator

3. Run the program:
python main.py

##  How It Works
* Enter your **weight (kg)**
* Enter your **height (m)**
* Click **Calculate Now**
* Your BMI score and category will be displayed instantly
